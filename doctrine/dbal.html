DBAL

Mapping Types

Strings

Type Since Maps SQL type to PHP type Description
string 2.0 VARCHAR string String with maximum length, for example 255.
text 2.0 CLOB string Strings without maximum length

Numbers

Type Since Maps SQL type to PHP type Description
integer 2.0 INT integer
smallint 2.0 SMALLINT integer
bigint 2.0 BIGINT string
decimal 2.0 DECIMAL string Restricted floats. Only works with a setlocale() configuration that uses decimal points.
float 2.0 FLOAT (double precision) double Only works with a setlocale() configuration that uses decimal points.

Dates and Times

Type Since Maps SQL type to PHP type Description
date 2.0 DATETIME DateTime
time 2.0 TIME DateTime
datetime 2.0 DATETIME/TIMESTAMP DateTime
datetimetz 2.0 Only PostgreSQL and Oracle support storing Timezone data.

Other Types

Type Since Maps SQL type to PHP type Description
boolean 2.0 boolean boolean
object 2.0 CLOB object Maps to a PHP object using serialize() and unserialize().
array 2.0 CLOB object Maps to a PHP object using serialize() and unserialize().
blob 2.2 BLOB resource stream
